\doxysubsection{Подробное описание}
Класс для сбора информации о текущей лексеме. 

\begin{DoxyAuthor}{Автор}
bonnyped 
\end{DoxyAuthor}
\begin{DoxyVersion}{Версия}
2.\+0 
\end{DoxyVersion}
\begin{DoxyDate}{Дата}
март 2024 года 
\end{DoxyDate}
\begin{DoxyWarning}{Предупреждения}
Пажалыстэ не душните, у меня дедлайн горит, поэтому возможно программа не работает или работает не до конца, а только до середины
\end{DoxyWarning}
Класс содержит в себе все необходимые для дальнешей работе с токенами информацией\+:
\begin{DoxyItemize}
\item Тип сущности
\item Число (если токен не является числом, поле по умолчанию рвно 0)
\item Лескема (если токен является числом, а не лексемой, поле по умолчанию равно 0)
\item Приоритетность (для операций и скобок)
\item Ассоциативность (для правоассоцитавных операций) 
\end{DoxyItemize}
