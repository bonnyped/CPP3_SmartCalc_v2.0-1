Данный класс решает следующие задачи\+:
\begin{DoxyItemize}
\item удаление пробелов из входящей строки (необходимо для последующей корректной обработки строки)
\item замена в необходимых местах знаков \char`\"{}-\/\char`\"{} на знак \char`\"{}$\sim$\char`\"{} (унарный минус)
\item перевод симоволов в нижний регистр
\item замены название текстовых функций и оператора mod 
\end{DoxyItemize}
