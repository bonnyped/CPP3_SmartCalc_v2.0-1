Parser::vector_reference s21::Parser::convertToPreRpn ( cstring_reference  input)

Функция проебразует строку в вектор токенов, определят сущность токена, обрабатывает числа с плавающей точки при помощи std:stod.

Функция решает задачу дополнительной более детальной и углубленной подготовки выражения, к дальнейшей обработке.

Вектор заполненных данными токенов.
